# Walimah Guest Seating System

This repository contains the seating lookup tool for Zakareeyah & Sana’s Walimah. 
The interface allows guests to type their name and instantly view their assigned table number. 
The system is designed to be fast, intuitive, and mobile‑friendly for use at the venue.

### Features
- Real‑time search with fuzzy matching
- Support for duplicate names (via middle initials / extended names)
- Clean, text‑only table number display
- Fully responsive layout for phones and tablets
- Simple data structure for updating guest names and table assignments

### Updating the Guest List
The guest list is not final yet.  
Names can be updated in the `PRESET` array inside `index.html` once confirmed.

### Purpose
This tool ensures a smooth and welcoming experience for guests as they arrive and get seated.
